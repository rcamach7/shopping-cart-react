# E-Commerce React Site 
Clean design of an e-commerce page with shopping-cart capabilities.

#### Local Installation & Running
```bash
git clone https://github.com/rcamach7/shopping-cart-react.git
cd shopping-cart-react
npm install
npm run start
```

#### Built Using:

- React Framework
  - react-router-dom
  - components
  - lifecycles
  - state management
  - event handlers
- PokeAPI
- Javascript
- HTML
- CSS
